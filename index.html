What is CSS?

  1. CSS stands for cascading sheets.

  2. CSS describe how HTML elements are to be displayed on screen, paper, or in media

  3. Css is not necessary CSS saves a lot of work. It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML document in 3 ways

  • Inline - by using the style attribute inside HTML element
  • Intermal - by using a style element in the head section
  • External- by using the link element to link to an external CSS files.
  • The most common way to add CSS, is to keep the styles in enternal CSS files. However, in this tutorial we will use inline and internal styles,because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference in that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by javaScript

In CSS, selector are used to target a specific element or range of element on a web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is a wide range of selevtors available. Two of the most commonly used are class and ID. Both are used to target element to which a style should be applied.
